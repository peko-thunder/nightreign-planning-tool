# プロジェクト

## 概要

2025年にフロム・ソフトウェアが開発したゲーム「ELDEN RING NIGHTREIGN」があります。
Youtubeなどのゲーム配信の企画に用いるためのツールを本プロジェクトで扱います。

このツールはWeb上で公開されます。

本プロジェクトのツールは、ゲームを直接ハックするものや攻略を有利にするものではありません。

## 企画ツール

### 夜渡りルーレット

マッチング成立後のキャラクター選択画面がベースです。
通常は、3人のプレイヤー(青, 赤, 緑)がキャラクターを任意に選択します。

事前にルーレットでプレイヤーのキャラクターを決定して、ゲームを開始するという使い方をします。

## 技術スタック

- TypeScript
- Tailwind CSS
